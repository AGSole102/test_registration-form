# Форма регистрации для тестового задания

## Требования

- Node.js `>= 20`
- npm `>= 10`

## Локальный запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить dev-сервер:

```bash
npm run dev
```

3. Открыть в браузере адрес из консоли (обычно [http://localhost:5173](http://localhost:5173)).

## Сборка и предпросмотр production-версии

Собрать проект:

```bash
npm run build
```

Запустить локальный предпросмотр сборки:

```bash
npm run preview
```

## Полезные команды

- Проверка линтером:

```bash
npm run lint
```
